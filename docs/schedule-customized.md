## 🎯 各ツールの使い分け

```
【Claude このチャット】
→ 主な思考・判断
→ 各エージェントへの詳しい指示

【NotebookLM】
→ 週報などの長いテキストを「音声」で理解したいとき
→ 複数のドキュメント（4つのエージェント定義）を一括分析したいとき
→ 「このドキュメント、要点は？」と聞く

【Gemini 3】
→ 「Twitter 投稿の案を 5 つ並行生成」など複数案を高速生成
→ Claude の回答と「別視点」での検証
→ データ分析補助

【ChatGPT】
→ 「ここ、別の言い方だと？」という言い換え
→ Claude と異なる視点での提案が欲しいとき
→ 最後の確認

推奨：
- **主軸**：Claude（ここ）で判断・指示
- **補助**：NotebookLM（音声・一括分析）+ Gemini（複数案生成）
- **確認**：ChatGPT（別視点）
```
